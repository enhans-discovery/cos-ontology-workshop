# 설비별 생산량 계산 공식

---

## 기본 지표

### 일일 처리량

설비의 하루 처리 가능 건수를 계산한다.

```
일일 처리량 = monthly_throughput / 22
```

- 22 = 월 영업일수 (주 5일 기준)
- 예시: M3(월 1800건) -> 일일 처리량 = 1800 / 22 = 약 82건

### 설비별 일일 처리량 현황

| 설비 | 방식 | 월간 처리량 | 일일 처리량 |
|------|------|-----------|-----------|
| M1 (A호기) | 매엽 | 800 | 약 36건 |
| M2 (B호기) | 매엽 | 750 | 약 34건 |
| M3 (C호기) | 윤전 | 1,800 | 약 82건 |
| M4 (D호기) | 윤전 | 1,500 | 약 68건 |
| M5 (E호기) | 윤전 | 2,000 | 약 91건 |

---

## 예상 처리 시간

현재 대기 작업을 모두 처리하는 데 걸리는 예상 일수를 계산한다.

```
예상 처리 일수 = queue_count / 일일 처리량
```

- 예시: M5(대기 110건, 일일 91건) -> 예상 처리 일수 = 110 / 91 = 약 1.2일

---

## 납기 위험 판정

주문별 납기 준수 가능 여부를 판정한다.

```
남은 영업일 = due_date - today (영업일 기준)
납기 위험 = 예상 처리 일수 > 남은 영업일
```

### 위험 등급

| 등급 | 조건 | 조치 |
|------|------|------|
| 정상 | 남은 영업일 > 예상 처리 일수 + 2일 | 정상 합대 진행 |
| 주의 | 남은 영업일 <= 예상 처리 일수 + 2일 | 합대 우선 배정 |
| 위험 | 남은 영업일 <= 예상 처리 일수 | 단독 판 우선 인쇄 또는 설비 재배정 |
| 긴급 | due_date - today <= 3일 AND status = 대기 | 즉시 단독 판 배정, 합대 제외 |

---

## 윤전 vs 매엽 효율 비교

- 윤전 설비는 롤 용지 연속 투입으로 대량 인쇄에 유리하다. 1,000부 이상 주문에서 매엽 대비 약 2~3배 빠르다.
- 매엽 설비는 낱장 투입 방식으로 소량 다품종에 유리하다. 500부 미만 주문에서 세팅 시간이 짧다.
- 동일 부수에서 윤전의 장점이 발휘되려면 최소 500부 이상이어야 한다.

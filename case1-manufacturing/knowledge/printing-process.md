# 인쇄 공정 5단계 절차

한빛인쇄의 교재 인쇄 공정은 5단계로 구성된다.

---

## 1단계: 주문 접수

고객(출판사, 학교)으로부터 인쇄 주문을 접수한다.

- Input: 고객 발주서 (교재명, 부수, 사양, 납기일)
- Output: print-order 데이터 1건 생성 (status=대기)
- 담당: 영업팀

주문 접수 시 확인 항목:
- 교재명, 인쇄 부수, 페이지 수
- 용지 종류 (아트지/모조지/백상지/코팅지)
- 제본 방식 (무선제본/중철제본/스프링제본)
- 완성 판형 크기 (210x297, 188x257, 148x210)
- 납기일

---

## 2단계: 조판 (프리프레스)

고객이 전달한 PDF 원고를 검수하고, 인쇄판에 배치한다.

- Input: PDF 원고 파일 + 주문 사양
- Output: 인쇄용 판 파일 (CTP 출력용)
- 담당: 조판팀

주요 작업:
- PDF 원고 색상/해상도/여백 검수
- 페이지 배열 (접지 순서에 맞게 재배치)
- 인쇄판 레이아웃 생성

---

## 3단계: 합대 (Ganging)

동일 사양(용지 종류 + 용지 규격 + 제본 방식)의 주문들을 하나의 판으로 묶어 용지 교체를 최소화한다.

- Input: 대기 상태의 주문 목록 + 설비 현황
- Output: gang-group 데이터 1건 생성 (주문 N건을 1그룹으로)
- 담당: 생산관리팀

합대 매칭 조건:
- 용지 종류가 동일해야 함
- 용지 규격이 동일해야 함
- 제본 방식이 동일해야 함
- 그룹당 최대 8건까지 묶을 수 있음
- 납기가 급한 주문은 단독 판으로 우선 배정

---

## 4단계: 인쇄

설비에서 실제 인쇄를 수행한다.

- Input: 합대된 인쇄판 + 용지
- Output: 인쇄된 낱장 (제본 전 상태)
- 담당: 인쇄팀

설비별 특성:
- 매엽 설비 (A호기, B호기): 낱장을 한 장씩 투입. 소량/다품종/소규격에 적합
- 윤전 설비 (C호기, D호기, E호기): 롤 용지를 연속 투입. 대량/대규격 생산에 적합

주문 status가 "대기"에서 "인쇄중"으로 변경된다.

---

## 5단계: 후공정

인쇄된 낱장을 제본하고 마무리 작업을 수행한다.

- Input: 인쇄된 낱장
- Output: 완성된 교재 (status=완료)
- 담당: 후공정팀

주요 작업:
- 재단: 인쇄물을 최종 판형 크기로 자르기
- 제본: 무선제본(풀 접착) / 중철제본(스테이플러) / 스프링제본(와이어)
- 코팅: 필요 시 표지 코팅
- 포장: 출고용 박스 포장

주문 status가 "인쇄중"에서 "완료"로 변경된다.

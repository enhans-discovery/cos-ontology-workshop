# 사전 준비 가이드

이 문서를 따라하면 COS 온톨로지 따라 만들기를 시작할 준비가 완료됩니다.

---

## Step 1: GitHub 레포지토리 다운로드

**상세 가이드**: `download-guide.md` 파일을 참고하세요.

**가장 간단한 방법**:
1. https://github.com/enhans-discovery/cos-ontology-workshop 에 접속합니다
2. 초록색 **<> Code** 버튼 > **Download ZIP** 클릭
3. 다운로드된 ZIP 파일 압축 해제

### 성공 확인
- 폴더 안에 `case1-manufacturing`, `case2-enterprise` 폴더가 보이면 성공

---

## Step 2: COS 로그인

### 1. 어떤 메뉴에 들어가야 하는가
웹 브라우저

### 2. 어떻게 수행하는가
1. 브라우저를 열고 [app.commerceos.ai](https://app.commerceos.ai)에 접속합니다
2. 강사가 배포한 종이에 적힌 **이메일**과 **비밀번호**로 로그인합니다
3. **로그인 후 반드시 새로고침(F5 또는 Cmd+R)을 한 번 해주세요**
4. 이미 개인별 Tenant가 배정되어 있으므로 별도 생성이 필요 없습니다

### 3. 성공했음을 어떻게 아는가
- COS 대시보드 화면이 나타나면 성공
- 좌측 메뉴에 Objects, Links, Knowledge, Agents 메뉴가 보이면 준비 완료

---

## 준비 완료 체크리스트

아래 항목을 모두 확인하세요:

- [ ] GitHub 레포지토리를 다운로드했다
- [ ] `case1-manufacturing/csv/` 폴더에 CSV 파일 3개가 있다
- [ ] COS에 로그인했다 (강사 배포 계정)
- [ ] 로그인 후 새로고침(F5)을 했다
- [ ] Tenant가 배정되어 있다 (화면 상단에 Tenant 이름 표시)

모두 완료되었으면, 케이스별 `guides/step1-*.md`부터 시작하세요.
